# Superstore Sales Analysis Dashboard

## Project Overview

This project analyzes a retail superstore sales dataset containing 9,800 transactions.

The objective was to identify sales trends, top-performing categories, customer segments, and regional sales performance using Microsoft Excel.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning

## Dashboard Features

* Total Sales KPI
* Total Orders KPI
* Total Customers KPI
* Total Categories KPI
* Sales by Region
* Sales by Category
* Sales by Sub-Category
* Monthly Sales Trend
* Sales by Customer Segment
* Interactive Slicers

## Key Insights

* Technology generated the highest sales revenue.
* The West region recorded the highest sales.
* Consumer customers contributed the largest share of sales.
* Phones and Chairs were among the top-performing product sub-categories.
